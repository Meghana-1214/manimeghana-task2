# Network Security & Scanning - Task 2

## Objective
Learn reconnaissance, scanning, and network traffic analysis using ethical hacking tools.

## Topics Covered
- Passive Recon (Whois, Nslookup, Google Dorking, Shodan)
- Active Recon (Ping Sweep, Banner Grabbing)
- Nmap Scanning
- Service & OS Detection
- Vulnerability Scanning (OpenVAS)
- Packet Analysis (Wireshark)
- SYN Flood Simulation

## Tools Used
- Nmap
- Wireshark
- OpenVAS
- hping3
- Metasploitable2

## Target System
- Metasploitable2 (Test VM)

## Outcome
Identified open ports, services, vulnerabilities, and analyzed network traffic.
