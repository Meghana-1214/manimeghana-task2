# Tools Used

## 1. Nmap
Used for port scanning, service detection, and OS detection.

## 2. OpenVAS
Used for vulnerability scanning and risk analysis.

## 3. Wireshark
Used to capture and analyze network packets.

Used to simulate SYN flood attack.

Used for passive reconnaissance.
